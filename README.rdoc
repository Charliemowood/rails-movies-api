
# Movie Rails API 

Movies API that returns and repond to JSON.

The INDEX route is located at localhost:3000/api/shows

To use run the app, run the following commands in Terminal:

1. Run the server:
  
  ```
  rails server
  ```

2. Migrate the database:

  ```
  rake db:migrate
  ```
  
3. Populate the database:
   ```
   rake db:seed
   ```
